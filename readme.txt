== Description ==

A simple, fast and optimized Wordpress themes for blogging and news.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Edrea in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. If you want to activated the slider you must have 1 tags called 'Trending' in your article or news (Optional), at least you have one to show up the slider.
5. The available menu is 2. first for header and second for footer. 
6. Edit The sidebar on single-post from widget setting

== Privacy ==
Edrea uses LocalStorage to save the setting when Dark Mode support is turned on or off.
LocalStorage is necessary for the setting to work and is only used when a user clicks on the Dark Mode button.
No data is saved in the database or transferred.
